[#{cl_sst_unsigned_ceil_div_id}]
= The `sst::unsigned_ceil_div` function

The `{cl_sst_unsigned_ceil_div}` function returns
`stem:[\lceil \mathtt{a} / \mathtt{b} \rceil]`
for various type combinations of `a` and `b`.
If `a` is negative, `b` is negative, or `b` is zero, the behavior is
undefined.

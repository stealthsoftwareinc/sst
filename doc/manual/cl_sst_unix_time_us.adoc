[[cl_sst_unix_time_us,sst::unix_time_us]]
= The `sst::unix_time_us` function

.{cpp}
[source,cpp]
----
#include <sst/time.h>
namespace sst {

inline std::chrono::microseconds::rep unix_time_us();

}
----

The `sst::unix_time_us` function returns the current tick count of the
`<<cl_sst_unix_time>>` clock in microseconds.
Note that the return type is an unspecified signed integer type whose
width is at least 55 bits (see `{ce_std_chrono_duration}` or
{cpp11sec}20.11.2).

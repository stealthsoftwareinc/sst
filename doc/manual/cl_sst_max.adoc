[#cl-sst-max]
= The `sst::max` function

.{cpp}
[source,cpp,subs="{sst_subs_source}"]
----
#include <link:{repo_browser_url}/src/c-cpp/include/sst/catalog/max.hpp[sst/catalog/max.hpp,window=_blank]>
// or:   <sst/algorithm.h>
namespace sst {

template<class T>
constexpr T const & max(T const & a, T const & b);

}
----

The `sst::max` function returns `a < b ? b : a`.

This function was added to the library because `{ce_std_max}` is not
`constexpr` before {cpp}14.

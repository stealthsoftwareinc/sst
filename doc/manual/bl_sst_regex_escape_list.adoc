[#bl-sst-regex-escape-list]
= The `sst_regex_escape_list` function

.Bash
[source,subs="{sst_subs_source}"]
----
include::partials/bl_include_prelude.adoc[]

{bl_sst_regex_escape_list} [<arg>]...
----

The `{bl_sst_regex_escape_list}` function outputs a
{bash_regular_expression} that matches any of the arguments.
If no arguments are given, the regular expression will never match.
The regular expression is always parenthesized.

The `sst::is_exact_width_integer` trait determines whether `T` is one of
the exact-width integer types, i.e., one of
`int8_t`,
`uint8_t`,
`int16_t`,
`uint16_t`,
`int32_t`,
`uint32_t`,
`int64_t`,
or
`uint64_t`,
ignoring any {cr_cv_qualifiers}.
